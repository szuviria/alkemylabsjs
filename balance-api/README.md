# Getting Started with Balance Api

Download the repository

### `install dependencies`

In the project directory, you run `npm install` to install dependencies

### `run api`

Runs the api in the development mode with `npm run dev`.\
Open [http://localhost:3001/api/operations](http://localhost:3001/operations) to view json operations in the browser.
Open [http://localhost:3001/api/user](http://localhost:3001/user) to view json user in the browser.
